**K-Beauty Store Website**

A modern and responsive e-commerce website dedicated to Korean skincare products. The platform features a carefully selected collection of the most popular and trusted K-beauty brands, offering customers a smooth and user-friendly shopping experience.

The project showcases clean UI/UX design, product categories, detailed product pages, and an attractive layout inspired by modern Korean beauty trends. To promote the brand, the website also highlights collaborations with influencers, actresses, and K-pop artists, creating a stronger connection between customers and the world of Korean skincare.

**Technologies Used:**

* HTML
* CSS
* JavaScript
* Responsive Design

This project was created to demonstrate front-end web development skills while building a visually appealing and realistic online skincare store.
